# Claude Skills

סקילים אישיים ל-Claude, בעברית.

## סקילים בריפו

### `shopify-israeli-store/` (גרסה 2)

סקיל לבניית ושיפור אתרי שופיפיי ברמה גבוהה לקהל ישראלי.

**תומך בשני מצבי עבודה:**
- **מצב בנייה** - בניית אתר חדש מאפס
- **מצב ביקורת** - סקירת אתר קיים עם המלצות שיפור

**מייצר:**
- אסטרטגיית מותג ופלטת עיצוב
- מבנה דפים עם קופי מוכן בעברית
- קוד Liquid/HTML/CSS מותאם לשופיפיי
- המלצות לאפליקציות ישראליות (תשלומים, משלוחים, חשבוניות)
- אסטרטגיית המרה מותאמת לקהל ישראלי
- ביקורות מפורטות של אתרים קיימים

**קבצי עזר:**

| קובץ | מתי נטען |
|---|---|
| `hebrew-communication.md` | תמיד - איך לתקשר עם המשתמש |
| `product-page-blueprint.md` | דפי מוצר |
| `homepage-blueprint.md` | דף בית |
| `hebrew-copywriting.md` | כל כתיבה בעברית לאתר |
| `israeli-market.md` | תשלומים, משלוחים, חוקים ישראליים |
| `visual-design-blueprint.md` | החלטות עיצוב ויזואליות |
| `audit-mode.md` | ביקורת של אתר קיים |
| `dropshipping-trust-guide.md` | אתרים שמוכרים מוצרי דרופשיפינג |

---

## איך להשתמש בסקיל

1. פתח צ'אט חדש ב-Claude
2. תגיד "בוא נבנה אתר שופיפיי" או שלח קישור לאתר קיים ותבקש ביקורת
3. Claude יטען את הסקיל ויפעל לפי התהליך המתאים

**דוגמאות להפעלה:**
- "אני רוצה לפתוח חנות שופיפיי" → מצב בנייה
- "תבקר את https://example.co.il/products/X" → מצב ביקורת
- "עזור לי לשפר את האתר שלי" → מצב ביקורת
- "בוא ננתח את דף המוצר הזה: [URL]" → מצב ביקורת
